PyBer cab analysis

Data provided by csv. The task was to showcase the relationship between the average fare ($) Per City, Total Number of Rides Per City, Total Number of Drivers Per City, City Type (Urban, Suburban, Rural) using a scatter plot. As well as the % of Total Fares by City Type ,% of Total Rides by City Type and ,% of Total Drivers by City Type using pie charts.

  • Date: October 2017
  • Featured Python Libraries: Pandas and matplotlib
  • Skills: Basic data cleaning and manipulation in pandas. Basic visualization and analysis using matplotlib.

Weather at the Equator

How does the weather change as we get closer to the equator? Data was collected from the OpenWeatherMap API and manipulated using python and pandas. Visualization were created for temperature, humidity, cloudiness, and wind speed of 700 randomly chosen countries around the world to determine the weather patterns around the equator.

  • Date: November 2017
  • Featured Python Libraries: numpy, pandas, matplotlib
  • Skills: Understand API documentation to retrieve data. Basic data cleaning and manipulation to create visualizations using matpoltlib.
